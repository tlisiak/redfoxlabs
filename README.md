# Red Fox Labs

Bay Area web design & local SEO for small businesses.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- React Router DOM
- Supabase

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build outputs to `dist/`.

## Deployment

Hosted on GitHub Pages with a custom domain (`redfoxlabs.co`). Deployments trigger automatically on every push to `main` via `.github/workflows/deploy.yml`.
